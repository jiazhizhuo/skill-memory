# skill-memory

OpenClaw 风格的三层记忆系统，基于 Mem0 + Qdrant 实现。

## 架构

```
工作记忆 (Working)     → 会话上下文，单次会话
中期记忆 (Mid-term)    → Daily notes 风格，7天过期
长期记忆 (Long-term)  → MEMORY.md 风格，永久存储
```

## 智能触发（自动执行）

### 自动 Search（回忆） - 无需手动调用

当检测到以下模式时，**自动搜索记忆**：

| 触发模式 | 示例 | 行为 |
|---------|------|------|
| 时间参照 | "之前"、"上次"、"记得" | 搜索相关记忆 |
| 偏好询问 | "我之前用什么"、"我的习惯是" | 搜索用户偏好 |
| 上下文缺失 | 用户描述模糊 | 搜索相关背景 |
| 会话开始 | 新会话启动 | 搜索最近相关记忆 |

### 自动 Save（保存） - 无需手动调用

当检测到以下模式时，**自动保存记忆**：

| 触发模式 | 示例 | 行为 |
|---------|------|------|
| 明确指令 | "记住"、"以后要记住" | 保存到长期记忆 |
| 偏好表达 | "我喜欢/讨厌"、"通常用" | 保存偏好 |
| 项目配置 | "这个项目用XX"、"配置在YY" | 保存项目上下文 |
| 重复模式 | 3次以上出现的信息 | 归纳保存 |

### 自动规整 - 可选触发

| 触发时机 | 行为 |
|---------|------|
| 会话结束时 | 总结本次会话要点 |
| 每日首次使用 | 回顾昨日记忆 |
| 记忆超过 100 条 | 触发整理优化 |

## 命令（显式调用）

```
//memory add <内容> --tier long  添加到长期记忆
//memory search <查询>           混合搜索
//memory list --tier mid         列出中期记忆
//memory today                   查看今日记忆
//memory long                    查看长期记忆
//memory stats                   查看统计
//memory delete <id>             删除记忆
```

## 前置条件

1. Qdrant: `docker run -d -p 6333:6333 qdrant/qdrant`
2. pip install -e .
3. 或使用 Qoder hooks 自动启动

## 搜索特性

- 向量搜索（语义理解）
- 关键词搜索（精确匹配）
- MMR 重排（多样性）
- 时间衰减（近期优先）

## 触发关键词参考

**Search 触发词：**
`之前`、`上次`、`记得`、`我之前`、`我的习惯`、`以前用`、`原来`、`曾经`

**Save 触发词：**
`记住`、`以后要`、`记住这个`、`这个配置`、`我用`、`偏好`、`习惯`
